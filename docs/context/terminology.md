# Terminology

- **plaintech** — the custom Hugo theme in `themes/plaintech/`
- **render hook** — Hugo template that overrides default markdown rendering for a specific element (images, code blocks)
- **content-file taxonomy** — tags implemented as markdown files in `content/tags/` rather than Hugo's built-in taxonomy system; each file's basename is the tag key
- **OG image** — OpenGraph preview image generated at build time by compositing text onto `assets/og_base.png`
- **prose** — CSS class applied to long-form content sections for typographic styling
